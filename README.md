# AI Srez Analysis

Streamlit-приложение для анализа результатов СОР и СОЧ любых форматов.

## Возможности
- Автоматически распознаёт столбцы даже если они названы по-разному
- Обрабатывает .xls, .xlsx, .csv
- Строит итоговую таблицу (качество, успеваемость)
- Создаёт диаграммы
- Генерирует выводы и рекомендации
- Устойчивая версия — не падает при ошибках

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
